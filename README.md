# EchoDesk Agent Enricher

Enriches Oakland County realtors with invalid emails from EchoDesk using MiRealSource, Realtor.com, and Zillow data.

## Setup
1. Save `echo-contacts-enricher.md` to `.claude/agents/`
2. Open Claude Code in this project folder
3. Apply EchoDesk filters: Invalid Email + agent + oakland county tags
